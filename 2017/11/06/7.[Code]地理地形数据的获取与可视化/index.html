1.3 Cartopy STAMEN 图形绘制

数据源stamen是开源的地理信息服务网站，提供道路、地形等多种数据信息。在其官方网页采用手动下方式，仅能以确定中心点和缩放比例的方式进行图片下载，不能实现任意区域内图形获取。此处参照Geology and Python博客中的相关教程，以中国全域为例，展示底图获取和绘制流程。

cartopy是用以分析并可视化地理数据的Python库。
